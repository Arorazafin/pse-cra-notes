Mesures organisationelles : DEVOPS ou DEVSECOPS

  • devops : c'est une culture de collaboration, et de communication et d'automatisation entre DEV et OPS
  • Méthode agile : c'est une phylosophie de gestion de projet.
  • Méthode scrum : c'est un framework pour l'application de la méthode agile
  • Les outils en ligne :
    • Confluence pour la redaction des documents fonctionnels,
    • Jira pour les suivi des taches : suivi de type Kanban board.
    • Gitlab pour les outils collaboratifs de développement. -devsecops$ : c'est d'integrer la securité des données dès le debut du projet. C'est une condition préalable pour commencer un projet.

Mesures pour la potection et la securité des données.

  • Anonymization des données par postgresql Anonymizer pour un jeu de test pour la formation sur l'application
  • mise en place de RGS (referentiel générale de sécurité - conforme aux normes européeens + cooeperation entre administrations)
  • mise en place de Plan de continuité d'activité en cas de risque majeur (c'est un document qui decrit les demarches à suivre)

Mesures pour la qualité interne du logiciel : mises en place des tests probants

  • les TU (J-unit)
  • les tests d'integration (interaction base de donnée - service - ihm)
  • les tests fonctionnels (selenium)
  • les tests de la navigation IHM (cucumber)
  • les tests de regression : l'intregration d'une nouvelle fonction n'entraine pas de bug dans l'application ?
  • les tests de performance (plusiurs utilisateurs en même temps sur le l'application)
  • les tests de vulnerabilité ou le test de securité

Mesures pour accelerer la transformation numérique

  • s'inscrire dans le cadre des orientations interministerierielles (mutualisation des couts + RGAA - reference commun)
  • Urbanisation du SI
  • Modernisation du SI de l'etat
  • Gestion electronique des documents

Mesures pour la reduction de la consommation de l'enerigie et de l'émission des gaz à effet de serre

  • mise en place de l'eco-conception logicielle
  • utilisation d'outils adaptable (Eclipse Eco4Eclipse - mesure l'impact environnemental du code source)
