Les principaux tests et les outils existants associés

Differents tests en informatique existent et des solutions existent pour performer ces tests

Les pricipaux tests

La mise en place de tests nécéssitent à définir les anomalies recherchés

  • Les tests unitaires sont appliqués sur les anomalies conceranant les méthodes codées
  • Les tests fonctionnels sont appliqués sur les anomalies sur les spécifications métiers.
  • Les tests de performance ou de charge sont appliqués sur anomalies liées au comportement du logiciel en condistion de production.
  • Les tests de vulnérabilités sont appliqués sur les anomalies concernant les anomalies sur la sécurités.

Les outils

  • Pour les tests unitaires, 'JUnit' peut-être appliqué
  • Pour les tests fonctionnels, 'Selenium' peut être utlisé
  • Les tests de performance ou de charge
  • Les tets de sécurité
